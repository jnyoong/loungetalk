# 라운지톡 — 서비스 기획문서

> 작성일: 2026-05-23 | 현재 단계: MVP 개발 중

---

## 1. 서비스 개요

### 한 줄 정의
서울 라운지바·클럽·루프탑바 정보를 한 곳에 모아 소비자와 사업주를 연결하는 나이트라이프 플랫폼

### 핵심 문제
- **소비자**: 오늘 밤 갈 곳을 찾으려면 수십 개 업장 인스타그램을 일일이 확인해야 함
- **사업주**: 인스타 외에 별도 마케팅 채널이 없고, 공연 정보 노출이 팔로워에게만 제한됨

### 해결 방식
- 서울 전역 업장 정보(입장료·장르·일정·라인업)를 한 플랫폼에 집약
- 업주 인스타그램을 자동 연동해 별도 업데이트 소요 제거 (Phase 2)
- 무보증금 예약으로 소비자 편의 제공

---

## 2. 타겟 사용자

### Primary — 서울 로컬 20~30대
- 주말 나이트라이프를 즐기는 직장인·대학생
- 여러 업장을 비교하고 싶지만 정보가 분산돼 있어 불편함을 느끼는 사람

### Secondary — 외국인 관광객 (K-culture)
- 한국 드라마·음악으로 서울 나이트라이프에 관심 있는 외국인
- 기존 해외 서비스(Discotech, RA)는 서울 커버리지 얕고 한국어 없음
- 영문 업장명·영문 소개 필드로 대응

### B2B — 업장 사업주
- 강남·이태원·홍대 라운지바·클럽 운영자
- 현재 마케팅 수단: 인스타그램 계정 1개가 전부

## 7. 단계별 로드맵

### Phase 1 — MVP (현재 완료)
- 소비자 앱: 탐색·상세·이벤트·즐겨찾기·무보증금 예약
- 사업주 앱: 업장 등록·이벤트 관리·예약 확인

### Phase 2 — 성장 (다음)
- 인스타그램 Graph API 자동 연동
- 카카오맵 지도 뷰
- 푸시 알림 (이벤트 알림, 예약 상태 알림)
- 업장 비교 기능

### Phase 3 — 수익화
- 실결제 예약 시스템
- 광고 상품 (상단 노출)
- 리뷰·평점
- 통계 대시보드 (사업주)
- 타 도시 확장 (부산)
